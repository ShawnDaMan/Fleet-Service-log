# Fleet Service Log

A web-based fleet maintenance tracking system with service logs, filtering, and cost analysis.

## Features
- Track vehicle service records with costs
- Filter by vehicle and date range
- View cumulative costs per vehicle
- Export/import data via CSV
- Visual charts for cost analysis

## Usage
Open `fleet_service_log.html` in your browser to start logging service records.

## Files
- `fleet_service_log.html` - Main service log interface
- `fleet_service_charts.html` - Visual cost analysis charts
- `assets/css/styles.css` - Shared styles
- `assets/js/fleet-service.js` - Application logic

## Deployment
Hosted on GitHub Pages. Data is stored locally in your browser's localStorage.
