# Fleet Service Log V2

A local-only, optimized version of the Fleet Service Log app for testing efficiency improvements.

## Features
- Local data storage (no Google Sheets)
- Optimized table rendering
- Efficient event delegation
- Chart rendering only on data change
- Modular JS and CSS
- No changes to production code

## Usage
Open `index.html` in your browser to use the app locally.

## Project Structure
- `index.html` - Main service log interface
- `assets/js/` - Application logic
- `assets/css/` - Shared styles
- `.github/copilot-instructions.md` - Project-specific Copilot instructions

## Development
This project is for local testing and optimization only. Production code remains unchanged.
